## Instalation on local
1. Start the repository
2. ` $ npm install `
3. Connect our repository to the database
4. ``` $ Sequelize: db:migration ``` 
5. ``` $ Sequelize: de:seed``` 
6. ``` $ npm run dev ``` 
7. ...
